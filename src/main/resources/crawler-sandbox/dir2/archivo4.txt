Este documento recoge el proceso de preparación del entorno y desarrollo de la Práctica 2, dividida en dos sub-prácticas: Práctica 2a - Introducción a Python y Práctica 2b - Acceso a datos desde Python. De la primera sub-práctica solo se documenta cómo se extendió el entorno instalado en las prácticas anteriores para trabajar con Python y Jupyter Notebook. De la segunda sub-práctica se documenta cómo se instaló la librería cx_Oracle y el desarrollo de los ejercicios propuestos.
 
Para realizar esta práctica aprovecho la instalación de IDEA Ultimate y hacerla extensible a Python. Es un IDE muy potente que, aunque fuese diseñado para trabajar con la plataforma Java principalmente, también permite extender su funcionalidad para soportar nuevas plataformas a través de plugins. Existe un plugin creado por sus desarrolladores que añade soporte para Python y Jupyter Notebook. Instalando la SDK de Python y la librería cx_Oracle tenemos en un solo IDE la solución para trabajar con Java, Python y SQL al estilo de SQL Developer.
