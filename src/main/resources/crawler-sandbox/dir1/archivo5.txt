Esta documentación recoge el desarrollo de los ejercicios 4.a) y 4.b) del guion Práctica 3 - PL/SQL. Previo al desarrollo de los ejercicios propuestos, he ejecutado los 7 scripts SQL que ejemplifican el uso de PL/SQL según el material visto en clase.
 
El entorno software utilizado para desarrollar esta práctica ha sido configurado durante las prácticas 0, 1 y 2, previas a este guion, y sin ninguna modificación adicional. Las características más destacables de IDEA Ultimate, IDE escogido sobre Eclipse y SQL Developer, son: reconocimiento del lenguaje PL/SQL de forma nativa para bases de datos de Oracle, autocompletado de código a través de IntelliSense, consola unificada para resultados tabulados de consultas y la salida DBMS_OUTPUT e integración en la misma ventana de un explorador de objetos de la base de datos y del editor de código, ofreciendo una experiencia continua para el desarrollo de programas PL/SQL. Para reproducir los resultados recogidos en este documento es necesario recrear las tablas EMPLEADOS y DEPARTAMENTOS a través del script 2 - Insertar.sql perteneciente al guion de la Práctica 0.
